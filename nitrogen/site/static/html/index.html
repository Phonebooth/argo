Welcome to the Argo login page!

Argo requires public key authentication.

function argo-login {
    # Creates a background ssh tunnel listening
    # locally onport 8000 and forwarding on the
    # remote host to port 8000

    ssh -L 127.0.0.1:8000:127.0.0.1:8000 argo.republicdev.info -Nf
}

function argo-logout {
    kill $(ps -ef | grep [a]rgo.republicdev.info | awk '{print $2}')
}

Once you've logged in, continue via your secure argo tunnel:
http://localhost:8000
